# Snapboard-OCR
Copy text in form images in your clipboard and turn them to paste-able text automatically in your clipboard

Follow [this](https://github.com/ShrutiHarrison/Image-to-Text-Convertor) repository by [Shruti Harrison](https://github.com/ShrutiHarrison) for details on OCR.

Follow [this](https://github.com/UB-Mannheim/tesseract/wiki) wiki for downloading and installing Tesseract OCR on Windows
